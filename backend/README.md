# Backend

Planned backend folder for the file-system browser API.

This should eventually contain:

- Express server setup
- SQLite or PostgreSQL database setup
- Node/file/folder API routes
- Search and suggestion routes
- Backend tests

Recommended implementation order:

1. Create database schema.
2. Seed the root folder.
3. Add folder listing endpoint.
4. Add create-folder endpoint.
5. Add create-file endpoint.
6. Add delete endpoint.
7. Add exact search endpoint.
8. Add prefix suggestions endpoint.
9. Add tests.
